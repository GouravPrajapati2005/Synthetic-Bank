# 🏦 Synthetic Bank Dashboard | Power BI Project

## 📌 Project Overview
The Synthetic Bank Dashboard is an end-to-end Power BI project designed to analyze and visualize banking operations using synthetic banking data. The purpose of this project is to transform large-scale financial datasets into actionable insights through interactive dashboards and KPI-driven reporting.

This dashboard focuses on customer accounts, card activities, loan distribution, account balances, and banking trends. It enables users to understand financial patterns and customer behavior through dynamic visualizations and data storytelling techniques.

---

## 🎯 Project Objectives

The primary objectives of this project were:

- Analyze customer account performance
- Track debit and credit card usage
- Monitor account balance distribution
- Identify customer growth patterns
- Study loan distribution and loan trends
- Analyze interest rates and loan categories
- Create interactive and user-friendly dashboards
- Build meaningful KPIs for decision-making
